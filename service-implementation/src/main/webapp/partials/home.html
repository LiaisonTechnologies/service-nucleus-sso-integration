Service Nucleus

A seed project for serious web apps, made from best-of-breed tech like Netflix's OSS Karyon, hardened to the tune of billions of daily requests, and AngularJS, a client MVC framework worthy of an Academy Award.

This project is meant to serve as a substantial starting point for enterprise and commercial web applications. It integrates Karyon and AngularJS with projects like Angular-UI, Angular-Strap, Bootstrap, and jQuery, along with examples of angular services, directives, controllers, and filters. Check out the out of the box components in the top navbar, browse example REST services, or perform a pseudo login for additional features...
